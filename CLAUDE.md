# Site Kayan Cassariego

## Stack
- Next.js + Tailwind CSS v4 + Framer Motion
- Deploy automático na Vercel (kayan-sitee.vercel.app)
- Repositório: github.com/Kayanlopes/Kayan-sitee

## Design
- Fontes: Poppins (headings, ExtraBold/SemiBold) + Overused Grotesk (body/nav)
- Cores: bg #0a0a0a, accent #fa7548, text gradient #8f8f8f → #404040, border rgba(255,255,255,0.08)
- Dark mode only

## Componentes customizados
- SplashScreen: animação de entrada com fundo #fa7548, slide up
- InteractiveGridPattern: grid interativo no hero
- HoverBorderGradient: botão CTA com borda gradiente animada
- FollowingPointer: cursor customizado com "Você está aqui"
- NoiseBackground: fundo com noise/gradientes animados no card Webbie RH
- StackedCards: cards de projetos que empilham como notificações ao scrollar
